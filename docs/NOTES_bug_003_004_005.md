# Catatan BUG-003, BUG-004, BUG-005 (butuh konfigurasi, bukan 1 file)

## BUG-003 — Foto profil gagal load (CORS "local address space")
Penyebab umum: gambar dari Supabase Storage diakses dari konteks dev/preview, atau host
Supabase belum diizinkan di `next.config`.

Cek/perbaiki:
1. `next.config.js|ts` → pastikan `images.remotePatterns` mengizinkan host Supabase-mu:
   ```js
   images: {
     remotePatterns: [
       { protocol: 'https', hostname: '<project-ref>.supabase.co', pathname: '/storage/v1/object/public/**' },
     ],
   }
   ```
2. Akses app lewat domain production (https), bukan IP/localhost — error "local address space"
   muncul saat browser memblok request ke private network dari origin publik.
3. Pastikan bucket avatar **public**, atau pakai signed URL.

## BUG-004 — WebSocket realtime gagal (wss://...supabase.co/realtime)
1. Pastikan `NEXT_PUBLIC_SUPABASE_URL` benar (https → otomatis jadi wss).
2. Supabase → Database → Realtime: aktifkan untuk tabel yang dipakai (mis. notifications, messages).
3. Sering hanya muncul di **preview deploy** Vercel; cek di domain production.
4. Kalau pakai free tier dan koneksi terbatas, throttle subscription (jangan subscribe di tiap komponen).

## BUG-005 — React hydration error (#425 & #422)
Penyebab: output render server ≠ client. Biasanya karena:
- Baca `localStorage` / `Date.now()` / `Math.random()` saat render awal (mis. komponen bahasa, streak, countdown).
- Tanggal di-format dengan locale berbeda di server vs client.

Pola perbaikan (tunda render dinamis sampai setelah mount):
```tsx
const [mounted, setMounted] = useState(false)
useEffect(() => setMounted(true), [])
if (!mounted) return <PlaceholderSkeleton />   // render stabil dulu
return <RealtimeThing />
```
Kandidat komponen yang perlu dicek: `LanguageProvider`/`LanguageSwitcher`, `UpgradeCountdownCard`,
`PresenceHeartbeat`, `DailyPulseCard`, dan apa pun yang menampilkan waktu relatif ("2 menit lalu").

> Catatan: fix i18n di paket ini (provider menunggu mount sebelum membaca localStorage) sudah
> mengurangi salah satu sumber hydration mismatch terkait bahasa.
